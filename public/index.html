Fanart.tv Help

Fallback Artwork

Plembfin uses fanart.tv as a fallback source for posters, backdrops, and logo art when TMDB doesn't have images for a title. A built-in project key is already configured — no key is required to use this feature.

Entering your own personal API key gives you higher rate limits and access to images uploaded by your fanart.tv account. To get a personal key:

  1. Create an account at fanart.tv
  2. Go to your Profile → API Key
  3. Copy your personal key and paste it above

Automatic Remote Backups

Mirror backups to Backblaze B2 cloud storage

Backblaze B2 Setup

Free 10 GB · ~5 min
  1. Create a free account at backblaze.com and enable B2 Cloud Storage.
  2. Buckets → Create a Bucket: pick a globally-unique name, set files Private. Note the Endpoint shown (e.g. s3.us-west-004.backblazeb2.com) — the middle part is your Region (us-west-004).
  3. Application Keys → Add a New Application Key: restrict it to that one bucket, then create. Copy the keyID and applicationKey — shown only once.
  4. Fill in Region, Bucket, keyID, applicationKey, Save, then Test. Tick Enabled and click Back Up Now.

The endpoint is derived from the region automatically. Backups are pruned to your retention count here too.

Restore Guide

                        How to restore
                      


                      
What a restore does

                      

                        The selected backup becomes the single source of truth. Plembfin runs an authoritative restore:
                      

                      
    
                        
  1. Wipes the local watch history, playstate and resume progress, and reloads them from the backup.
    2. 
                        
  2. Pushes that state out to every connected app (Plex, Emby, Jellyfin).
    3. 
                        
  3. Stamps a restore watermark so the background sync can never re-import the apps' state afterwards.
    4. 
                      

                      

                        While the restore is running, the cron sync and incoming webhooks are ignored, so nothing is written to watch history mid-restore. Watch dates are always preserved — an item already watched on an earlier date is never re-added as watched today.
                      


                      
Reconcile tracked items

                      

                        Pushes only the items the backup knows about. Fast. Any extra items the apps have marked watched but the backup never tracked are left untouched. Choose this for a normal point-in-time restore.
                      


                      
Full wipe then push

                      

                        First marks every currently-watched item on each app as unwatched, then re-applies only the backup's watched set, so the apps end up matching the backup exactly. Slower (it touches every watched item on every app). Choose this when the apps contain watched state you want removed.
                      


                      
Restore onto New Instance

                      
    
                        
  1. Install a fresh Plembfin.
    2. 
                        
  2. Go to Backups → Settings and configure remote destinations.
    3. 
                        
  3. Go to Backups → Restore and click "Watch History Wipe / Restore" on a backup from your remote destination.

Sync Issues Help

                        Understanding Issues
                      


                      

                        What are Sync Issues?
                        

                          This section shows any problems with propagating watched states between your media servers. Click on any issue to view detailed telemetry about what went wrong.
                        

                      


                      

                        Issue Categories
                        

                          Missing Dispatch Telemetry: No logging data recorded. These likely synced successfully but the logs are missing. Safe to clear.
                        

                        

                          Plex Match Issues: Plex couldn't find the media item. Check Plex metadata and external IDs.
                        

                        

                          Emby/Jellyfin Match Issues: Plex found it but Emby/Jellyfin couldn't. Check their metadata and IDs.
                        

                        

                          Unresolved Sync Issues: Other sync problems detected. Click to view telemetry for error details.
                        

                      


                      

                        How to Fix
                        

                          Click "Retry" on an issue to re-attempt the sync and see detailed error messages. Click "Fix" to manually adjust the match if the item wasn't found.
                        

                      

                    

                  

                


                
                

                  

                    

                      
                      

                        

                      

                    

                  


                  

                    

                      

                        
Sync History Help

                        Recent Propagations
                      


                      

                        What is Sync History?
                        

                          Recent watched-state propagation attempts. Shows source, action, timestamp, and target sync results for every automatic and manual sync.
                        

                      


                      

                        How Sync Works
                        

                          When you mark something as watched (on any platform), Plembfin automatically propagates that state to your other configured servers every 60 seconds.
                        

                        

                          Resume positions sync similarly. Use "Force Sync" to immediately push any pending updates without waiting.

Sync Tools Help

                        Manual Control
                      


                      

                        Run Cron Sync: Executes the scheduled sync job immediately, checking for outstanding propagations and updating resume positions across all platforms.
                      

                      

                        Force Sync: Forcefully propagates all pending watched-state changes to your other platforms right away, bypassing the 60-second scheduled interval. Use this to immediately sync a watch or unwatched state.
                      

                      

                        Stop Sync: Halts a currently running force sync operation. Appears only when a sync is actively running.
                      

                      

                        Refresh: Reloads the sync issues list to check for any new or resolved issues without waiting for the next scheduled check.

Media Pages

                        What each toggle controls
                      

                      

                        Show Logo Art — Displays the title's logo image (fetched from TMDB) in the hero area at the top of the page. When off, the plain text title is shown instead.
                      

                      

                        Show Cast Members — Horizontal scrolling row of cast cards with headshots and character names. Clicking a card opens the person's profile.
                      

                      

                        Show Trailers & Clips — Embeddable YouTube trailer and teaser thumbnails sourced from TMDB. Requires a TMDB API key.
                      

                      

                        Show Reviews — Up to three critic or user reviews pulled from TMDB. Requires a TMDB API key.
                      

                      

                        Show Images Section — A scrollable gallery of backdrops and stills for the title from TMDB.
                      

                      

                        Show Related Shows — A row of similar titles recommended by TMDB. Clicking one navigates directly to that show's page.
                      

                    


                    

                      

                        
Notes

                        Behaviour & storage
                      

                      

                        Trailers, Reviews, Images, Cast, and Related content all require a configured TMDB API key under Settings → API Keys.
